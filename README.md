# SVG City Map → STL

Конвертирует SVG-карту города в 3D STL с экструдированными дорогами и зданиями.

## Установка

```bash
pip install -r requirements.txt
```

## Использование

```bash
python svg_to_stl.py карта.svg карта.stl
```

### Параметры

| Флаг | По умолчанию | Описание |
|------|-------------|----------|
| `--base-height` | 1.0 | Толщина подложки (мм) |
| `--road-height` | 0.5 | Высота дорог над подложкой (мм) |
| `--building-height` | 5.0 | Высота зданий (мм) |
| `--road-buffer` | 3.0 | Полуширина дорог в единицах SVG |

Пример с кастомными высотами:
```bash
python svg_to_stl.py карта.svg карта.stl --building-height 8 --road-height 1
```

## Как работает определение слоёв

Скрипт смотрит на атрибуты `id`, `class`, `fill` каждого элемента SVG:

- **Здания** — содержат: `building`, `house`, `block`, `здани`, `дом`, `квартал`
- **Дороги** — содержат: `road`, `street`, `way`, `highway`, `дорог`, `улиц`
- **Прочее** (парки, вода) — всё остальное, экструдируется на 0.2 мм

Если карта не размечена — всё попадёт в «прочее». В этом случае откройте SVG в Inkscape и назовите слои/группы согласно ключевым словам выше.
